Here's some stuff I made recently.

Android and iOS projects you can find in the App Store.

Streambels (AllConnect)

Stream what's on your phone or tablet to any speakers, smart TV, set-top box, or any other connected device in your home.

AllConnect SDK

AllConnect technology provides a unified API that allows you to build mobile apps that communicate with billions of devices.

InPoint

The indoor positioning project InPoint was funded by Ericsson, organized by Design Factory, Aalto University. Our team built an indoor positioning prototype using Wi-Fi fingerprint technology. The outcome is an Android application, an Android calibration tool, and a positioning engine running on the server.

UPnP Scanner

UPnP Scanner is an Android App which can discover any UPnP and DIAL devices present in the local network.

Tethering Shortcut

Tethering Shortcut is an Android App that gives you a much simpler way to create your own WiFi spot on your phone.

Screen Recorder for Root

Screen recorder is an Android tool to record your phone screen as MP4 video to your SD card. *Root required.

Site Knows

Site Knows is an unfinished project which utilizes indoor positioning technology to provide services for indoor shopping or navigation use cases.

Academic Projects

And here are some school project work I have done.

Network Programming

Project work for network programming course at Aalto University. It includes an HTTP client, an HTTP server, and an HTTP proxy. All are written in C and POSIX APIs.

Poker Manager

Project work for C++ course at Aalto University. The program consists of a history file parser, a SQLite database, a statistic module, and user interface written with Qt.

Car Project

Project work for Embedded System course at Aalto University. The program drives a racing car and navigates a given route autonomously. The program is written in C and cross-compiled to a micro controller.

Code Cleaner

Homework for UNIX Application Programming course. This is a program to remove C/C++ comments and empty lines in several files at the same time. Every file is processed in a separate process. The cleaned code is stored to another file named filename.cleaned. The log information is stored to a log file. It is written in C and POSIX APIs.

Print Capture

Homework for UNIX Application Programming course. This program offers a function named redirect(char *F_name) to capture the standard output stream and print the content to a file, whose file name can be defined in the old program. The child processes and threads spawned by the programs are also supported. However, STDERR message is not redirected and should be left untouched. It is written in C and POSIX APIs.
